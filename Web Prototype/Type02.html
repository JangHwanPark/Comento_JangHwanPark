나의 소개

안녕하세요, 저는 웹 개발자 홍길동입니다.

프론트엔드와 백엔드 개발 모두에 경험이 있으며, 사용자 중심의 디자인과 최적화된 개발을 목표로 합니다. 2018년부터 웹 개발을 시작하여 다양한 프로젝트를 경험했습니다.

저는 새로운 기술을 배우는 것을 좋아하며, 문제 해결을 위한 창의적인 접근 방식을 항상 고민합니다. 팀 프로젝트에서는 소통과 협업을 중요시하며, 개인 프로젝트에서는 완성도 높은 결과물을 위해 노력합니다.

취미로는 여행, 사진 촬영, 독서를 즐기며 이러한 경험들이 제 디자인과 개발 능력에 영감을 줍니다.

기술 스택

프론트엔드

HTML, CSS, JavaScript를 기반으로 사용자 친화적인 인터페이스를 구현합니다.

React, Vue.js와 같은 프레임워크를 활용한 SPA 개발 경험이 있습니다.

백엔드

Node.js, Express를 이용한 서버 구축 및 API 개발이 가능합니다.

MongoDB, MySQL 등의 데이터베이스 설계 및 관리 경험이 있습니다.

디자인

Figma, Adobe XD를 활용한 UI/UX 디자인 경험이 있습니다.

반응형 웹 디자인과 접근성을 고려한 디자인을 구현합니다.

프로젝트

온라인 쇼핑몰

React와 Node.js를 활용한 풀스택 이커머스 웹사이트입니다. 사용자 인증, 결제 시스템, 상품 관리 기능을 구현했습니다.

React Node.js MongoDB Express

여행 블로그

Vue.js와 Firebase를 활용한 여행 경험 공유 플랫폼입니다. 지도 API를 활용한 위치 기반 서비스를 제공합니다.

Vue.js Firebase Google Maps API

할 일 관리 앱

React Native로 개발한 크로스 플랫폼 모바일 앱입니다. 사용자 맞춤형 알림 기능과 캘린더 통합 기능을 구현했습니다.

React Native Redux Expo
